**OpenWISP-RADIUS** provides an admin interface to a `freeradius
<https://freeradius.org/>`_ database and offers features that are common
in WiFi and ISP deployments.

**Need a quick overview?** `Try the OpenWISP Demo
<https://openwisp.org/demo.html>`_.

.. note::

    If you're building a public wifi service, we suggest to take a look at
    `openwisp-wifi-login-pages
    <https://github.com/openwisp/openwisp-wifi-login-pages>`_, which is
    built to work with openwisp-radius.
